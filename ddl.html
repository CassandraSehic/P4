Using DDL in Microsoft Access

Microsoft Access is a database management system, that deals with relationships between data and helps you store information for referencing or analyzing later. Data Definition Language or (DDL) is a standard of commands you can use to manipulate a database. Microsoft Access is designed so that it is not necessary to use commands to change the database. However, it may be useful to incorporate DDL when using Microsoft Access. Here is how:

  1. Open a database and choose to create a query. Then close the show table window.
  2. Choose SQL view and there you will be able to type a DDL command or paste one in.
  3. If you want to create two tables and their indexes, use the function CreateTableDDL()
  4. If you want to change the type or a size of a field, this is easy to do with DDL. Use the function ModifyFieldDDL().
