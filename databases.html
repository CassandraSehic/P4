Practical Databases


Databases are essesntial to the current state of the web. Almost every Web 2.0 application uses databases to store information. It usually works like this; SQL databases are connected tables consisting of predefined rows and columns with each row having its own unique primary key. The blockchain is different than a database but understanding databases is important to understanding how the blockchain stores transactions. Everyone with internet connection has access to the blockchain, so we can find any transaction with its "primary Key", but instead each block has its own unique hash.
