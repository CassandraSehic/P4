Generating Key Pairs with the PGP Command Line

  1. Open a command shell.
  2. Type pgp --gen-key [user ID] --key-type [key type] --bits [bits #] --passphrase [passphrase] replacing the square brackets with the correct information for your key pair.
  3. Press ENTER. This will generate your key pair and return your key ID and a message that it was generated successfully.
  4. To display your new key pair you can type pgp --list-keys the shorthand version pgp -l will also work.
  5. You should display the key before you export it to be sure it is the right one.
  6. To export the key, type pgp --export (input) where input can be either the user ID or key ID.
  7. You can follow the same proccess to import a public key. Just type pgp --import (input).
  8. Finally, if you want to delete a key pair from your keyring, use the command pgp --remove (input)
